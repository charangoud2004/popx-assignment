# PopX-Assignment

A modern, mobile-first React application built for the PopX qualifier task. The project features a pixel-perfect implementation of the provided design with seamless navigation between the Sign In, Sign Up, and Profile screens.

## Live Demo
https://jocular-churros-6b93e9.netlify.app/

## Key Features
* **Mobile-First Responsive Design:** Optimized for mobile screens and perfectly centered on the webpage for desktop viewing.
* **Seamless Navigation:** Integrated with React Router DOM for efficient transitions between pages.
* **Pixel-Perfect UI:** Closely follows the provided design brief with a clean and well-formatted code structure.
* **Fast Development:** Built using Vite for a high-performance development experience.

## Tech Stack
* React (Frontend Library)
* Vite (Build Tool)
* React Router DOM (Navigation)
* Tailwind CSS (Styling)

## Folder Breakdown
* `/src/pages`: Contains Home, Login, Register, and ProfilePage views.
* `/src/routes`: Centralized routing configuration for the application.
* `/src/assets`: Image assets and icons used in the UI.

## Getting Started
1. Clone the repository.
2. Install dependencies.
3. Run the development server.
